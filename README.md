# 停车场管理系统 java22

## 项目概述

停车场管理系统是一款基于B/S架构、MVC模式的Web应用程序，采用前后端分离的开发模式，结合Java和Vue技术栈，实现了一套高效便捷的停车场管理解决方案。

## 技术栈

- **后端技术**：Java、SpringBoot、Mybatis-Plus
- **前端技术**：Vue
- **数据库**：Mysql
- **开发工具**：IDEA、JDK1.8、Maven

## 系统角色

- **普通用户**：具备个人信息管理、预定停车位、缴费信息管理等权限。
- **管理员用户**：具备用户信息管理、车辆信息管理、停泊车辆管理、车辆进出管理等权限。

## 运行环境

- **系统环境**：Windows

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理
